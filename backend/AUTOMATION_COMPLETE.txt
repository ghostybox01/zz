🚨 TROUBLESHOOTING

"Go not found" (Windows):
  → Install: https://go.dev/dl/go1.21.6.windows-amd64.msi
  → Restart terminal
  → Run START.bat again

"SSH connection failed":
  → Test: ssh -i /root/ssh/1 root@<VPS-IP>
  → Fix permissions: chmod 600 /root/ssh/1
  → Check: cat ssh_config.json

"No servers reachable":
  → Check server_ips.txt has correct IPs
  → Verify SSH key copied to servers
  → Test each server manually

"targets.txt not found":
  → Create: echo "https://example.com" > targets.txt

Force clean rebuild:
  → rm -f raven-scanner scanner_package.tar.gz
  → python start.py --rebuild
